# プライバシーポリシー自動生成ツール（for Japan）

このツールは、日本国内のサイト運営者向けに  
プライバシーポリシーを自動生成するためのサーバーレスアプリケーションです。

フォームに入力された情報は保存されず、  
生成されたHTMLはその場でブラウザに返されます。  
個人情報保護法および関連ガイドラインに基づき、  
日本国内でのWebサイト運営を支援することを目的としています。

ファイル / フォルダ	内容
api/generate.js	サーバーレス関数の本体。フォームデータを受け取り、テンプレート統合関数を呼び出します。ログ出力や保存は行いません。
api/policy-template.js	各セクションテンプレートを条件に応じて組み立て、完成したHTMLを生成します。テンプレート構成の指揮的役割。
api/config/sections.js	使用するセクション（項目）の順序と条件を定義します。法改正時の追加・削除はこのファイルで行います。
api/templates/	各項目（個人情報の取得方法、利用目的、Cookie、第三者提供など）の文面テンプレートを格納。1ファイル＝1セクション単位で独立しています。
index.html	UIkit3を使用した入力フォーム。利用者が理解しやすいよう、各入力欄に説明文を付与しています。生成結果をその場で表示します。
vercel.json	Vercel デプロイ設定。サーバーレス関数のルート指定やビルド設定を含みます。
README.md	このドキュメント。プロジェクト概要、セットアップ方法、更新履歴を管理します。
